## How to Use

1. **Add Your Invitation Image:**
   - Place your invitation image in the same folder as `index.html`
   - Rename it to `invitation-image.jpg` OR
   - Update the image source in the invitation section (around line 158) with your image filename:
     ```html
     <img src="your-image-name.jpg" alt="Birthday Invitation" ...>
     ```

2. **Update Venue Information:**
   - Edit the venue section with your actual:
     - Venue name
     - Address
     - Date
     - Time
   - Update the Google Maps link in the "Join Us" section

3. **Add Photos:**
   - Replace the placeholder emojis in the Memories section with actual photos
   - Update the "Our Little Princess" section with Deshna's photo (replace the ðŸ‘¸ emoji)

4. **Customize Details:**
   - Update dates, times, and locations throughout the page
   - All text is easy to find and update
